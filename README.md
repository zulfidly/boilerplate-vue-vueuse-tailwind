[Github](https://github.com/zulfidly/boilerplate-vue-vueuse-tailwind) 

- ```npm init vue@latest```
- ```cd v```
- ```npm install```
- ```npm run dev```
- ```npm i @vueuse/core```


- ```npm install -D tailwindcss postcss autoprefixer```
- ```npx tailwindcss init -p```
- see ```content: ["./src/**/*.{vue,js}"],``` in ```tailwind.config.js```

```
@tailwind base;
@tailwind components;
@tailwind utilities;
```
